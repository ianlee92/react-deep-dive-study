## ✅ Git Scenario

- 먼저, `git clone`을 이용하여 repo를 클론해주세요.
- 본인의 feature branch를 생성하여 커밋을 올려주세요.
  - `git add .`
  - `git commit -m "커밋 메시지"`
  - `git branch -M 브랜치명`
  - `git push -u origin 브랜치명`
- main branch로 PR(Pull request)를 올려주세요.
